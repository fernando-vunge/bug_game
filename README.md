#### INSTALACAO

**requisitos**
    
    - Git
    - Python3
    - pip 

*Siga esses passos para instalar esse prototipo no seu computador*

*clone o repositorio git!*

```
    git clone https://github.com/fernando-vunge/bug_game.git
```

*entre na pasta!*

```
    cd bug_game/
```
*use o pip para instalar a biblioteca ***pygame*** !*

```
   pip install pygame
```
